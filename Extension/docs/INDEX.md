## 🎓 Learning Paths

### Path 1: Quick Start (15 minutes)
1. Read [README.md](README.md) (5 min) - Overview
2. Configure [config.js](../config.js) (2 min) - Set website URL
3. Follow [INSTALLATION_GUIDE.md](INSTALLATION_GUIDE.md) (5 min) - Install
4. Test basic scanning (3 min) - Click extension icon

### Path 2: Complete Setup (30 minutes)
1. Read [README.md](README.md) (10 min) - Full features
2. Read [CONFIGURATION.md](CONFIGURATION.md) (10 min) - All options
3. Follow [INSTALLATION_GUIDE.md](INSTALLATION_GUIDE.md) (5 min) - Install
4. Follow [TESTING_GUIDE.md](TESTING_GUIDE.md) (5 min) - Basic tests

### Path 3: Advanced User (1 hour)
1. Read all documentation files
2. Complete full testing checklist
3. Customize icons and styling
4. Test in multiple browsers
5. Configure for production deployment

### Path 4: Developer (2+ hours)
1. Study [FOLDER_STRUCTURE.md](FOLDER_STRUCTURE.md) - Understand code
2. Review all `.js` files in root
3. Read threat detection algorithm
4. Modify and test changes
5. Create custom features
