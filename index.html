About PhishGuard

Protecting users from phishing threats with intelligent URL analysis

Our Mission

PhishGuard was built to make the internet safer for everyone. We provide real-time protection against phishing attacks through advanced URL analysis, helping users verify links before they click.

How We Work

Our intelligent scanning engine analyzes URLs across multiple dimensions:

  • Domain reputation and SSL certificate validation
  • Detection of lookalike domains and suspicious TLDs
  • Identification of phishing keywords and patterns
  • Real-time threat intelligence integration

Open Source Security

PhishGuard is committed to transparency and community-driven security. Our detection algorithms are continuously improved to stay ahead of evolving phishing tactics.

API Documentation

Integrate PhishGuard threat detection into your applications

🚀 Getting Started

The PhishGuard API allows you to programmatically scan URLs and receive threat assessments. Perfect for integrating security checks into your applications, browser extensions, or security tools.

📋 API Endpoint

POST https://api.phishguard.com/v1/scan

Request Body:

{
  "url": "https://example.com",
  "api_key": "your_api_key_here"
}

📤 Response Format

{
  "url": "https://example.com",
  "score": 94,
  "status": "Safe",
  "ssl": "Valid",
  "reputation": "Clean",
  "risk": "Low",
  "flags": [],
  "timestamp": "2024-12-11T12:00:00Z"
}

🔑 Authentication

API access requires an API key. To request access:

  1. Create a PhishGuard account
  2. Contact us at api@phishguard.com
  3. Describe your use case and expected volume
  4. Receive your API key within 1-3 business days

📊 Rate Limits

  • Free Tier: 100 requests/day
  • Basic: 1,000 requests/day ($29/month)
  • Pro: 10,000 requests/day ($99/month)
  • Enterprise: Custom limits (contact us)
